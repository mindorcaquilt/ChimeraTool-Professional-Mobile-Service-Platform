# ChimeraTool — Full Feature Reference

## Samsung Operations

### Flashing
- Full 4-file firmware flash (BL + AP + CP + CSC)
- Selective partition flash (individual partitions)
- Emergency flash via EDL (Qualcomm) and UART
- OTA package sideload
- Bootloader downgrade (model-dependent)

### Unlock
- SIM / Network unlock — all regions
- Country lock removal (South Korea, US carriers, Japan)
- Sprint / T-Mobile / AT&T unlock
- MetroPCS carrier removal

### Security
- FRP (Factory Reset Protection) bypass — Android 8–14
- Reactivation Lock (KG) removal
- Samsung Account removal from S7–S24 series
- Screen lock removal (pattern, PIN, password, fingerprint enrolled data)
- Knox counter reset (model-dependent)

### Repair
- IMEI 1 and IMEI 2 repair
- EFS partition backup / restore
- Bluetooth MAC address fix
- WiFi MAC address fix
- Serial number repair
- Build date and model string repair

### Diagnostics (free, no credits)
- Battery health and cycle count
- Display burn-in test
- Touchscreen uniformity scan
- Camera module test (front/rear/ultra-wide/telephoto)
- Speaker and microphone test
- Proximity and ambient light sensor check
